## Question
איזה עקרון יישמנו ב-weather monitoring system כאשר השתמשנו ב-reflection

### Options
- OCP
- LSP
- SRP
- ISP

## Answer
Using reflection in a weather monitoring system, especially for dynamically loading or instantiating components, aligns with the Open/Closed Principle (OCP). OCP states that software entities should be open for extension, but closed for modification. Reflection allows adding new functionality (extensions) without altering existing code (modifications).
