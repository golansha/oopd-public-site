## Question
מערכת לניהול כח אדם בצבא קבעה תקן עבור הפיקוד על כיתת חיילים. הפונקציה `fixSquad` מקבלת כיתה (`Squad`) שיש בה חיילים ומפקדים ומתקנת את המבנה שלה. להלן תנאי הקדם והבתר של הפונקציה:
תנאי קדם: לכיתה יש לכל הפחות 10 חיילים ולכל היותר 15.
תנאי בתר: לכיתה יש לכל היותר מפקד כיתה אחד.
ישנן מספר תתי מחלקות שבהן נדרסה הפונקציה.
1.  תנאי קדם: לכיתה יש בדיוק 12 חיילים. תנאי בתר: לכיתה יש בדיוק 2 מפקדים.
2.  תנאי קדם: לכיתה יש לכל הפחות 9 חיילים. תנאי בתר: לכיתה יש בדיוק מפקד אחד.
3.  תנאי קדם: לכיתה יש לכל הפחות 12 חיילים ולכל היותר 15. תנאי בתר: הכיתה ללא מפקד.
4.  תנאי קדם: לכיתה יש לכל היותר 16 חיילים. תנאי בתר: הכיתה ללא מפקד.
אילו מבין המקרים שתוארו מותרים על פי LSP:

### Options
- 2+4
- 2+3
- 3+4
- 1+2

## Answer
האפשרות הנכונה היא "2+4". עקרון ההחלפה של ליסקוב (LSP) קובע שתנאי קדם בתת-מחלקה יכולים להיות שווים או חלשים יותר (פחות מגבילים) מתנאי הקדם במחלקת האב, ותנאי בתר בתת-מחלקה יכולים להיות שווים או חזקים יותר (יותר מגבילים) מתנאי הבתר במחלקת האב.

ננתח כל מקרה:
*   **תנאי אב:** קדם: `10 <= חיילים <= 15`. בתר: `מפקדים <= 1`.

*   **מקרה 1:**
    *   קדם: `חיילים == 12`. זהו תנאי חזק יותר (מגביל יותר) מתנאי האב (`10 <= 12 <= 15` הוא נכון, אך `== 12` ספציפי יותר). **מפר את LSP.**
    *   בתר: `מפקדים == 2`. זהו תנאי חלש יותר (מאפשר יותר, או מפר את תנאי האב `מפקדים <= 1`). **מפר את LSP.**

*   **מקרה 2:**
    *   קדם: `חיילים >= 9`. זהו תנאי חלש יותר (מאפשר `9` חיילים, בניגוד למינימום `10` באב). **עובר את LSP.**
    *   בתר: `מפקדים == 1`. זהו תנאי חזק יותר (מגביל יותר מ-`מפקדים <= 1`). **עובר את LSP.**

*   **מקרה 3:**
    *   קדם: `12 <= חיילים <= 15`. זהו תנאי חזק יותר (מגביל יותר מ-`10 <= חיילים`). **מפר את LSP.**
    *   בתר: `מפקדים == 0`. זהו תנאי חזק יותר (מגביל יותר מ-`מפקדים <= 1`). **עובר את LSP.**

*   **מקרה 4:**
    *   קדם: `חיילים <= 16`. זהו תנאי חלש יותר (מאפשר `0` חיילים, בניגוד למינימום `10` באב). **עובר את LSP.**
    *   בתר: `מפקדים == 0`. זהו תנאי חזק יותר (מגביל יותר מ-`מפקדים <= 1`). **עובר את LSP.**

לכן, רק מקרים 2 ו-4 עומדים בעקרון LSP.
